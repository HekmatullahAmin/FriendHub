# FriendHub

FriendHub is an Instagram-like application that allows users to share photos, follow friends, and like and comment on posts. It is built using Cloud Firestore as its backend and follows the MVVM architecture pattern. The app is developed using Kotlin. 

## Features

- User authentication: Securely authenticate users using Firebase Authentication.
- Share photos and videos: Capture and share your moments with friends and followers. 
- Follow friends: Stay connected with your friends and see their latest posts. 
- Like and comment on posts: Show your appreciation for posts and engage with other users. 
- Edit profile: Customize your profile with a profile picture, bio, and name. 
- Send friend requests: Expand your network by connecting with new people. 

## Technologies Used

- Firebase Authentication: For user authentication and authorization.
- Android Studio: The app is developed using Android Studio.
- Cloud Firestore: For storing and managing user data and posts. 
- MVVM architecture: For a clean and maintainable codebase. 
- Kotlin: The primary programming language used for app development.
